アカウント削除について

本ページは、Google Play Console の「アカウント削除URL」として公開するものです。O-GIRI:RANK（以下「本サービス」）におけるアカウント削除の方法と、削除時のデータの取り扱いについて説明します。

本ページからアカウントを削除することはできません。アカウントの削除は、本サービスのモバイルアプリ内からのみ行えます。

アカウント削除方法

本サービスでは、Googleアカウントでログインした利用者が、アプリ内からいつでもアカウントを削除できます。

ゲスト利用（未登録）の場合は、設定画面にアカウント削除の項目は表示されません。

操作手順

  1. 本サービスのアプリを起動し、Googleアカウントでログインします。
  2. プロフィール画面右上の設定アイコンをタップし、「設定」画面を開きます。
  3. 設定画面の下部にある「アカウントを削除」をタップします。
  4. 確認ダイアログの内容を確認し、「削除する」をタップします。
  5. Googleアカウントでの再認証を行います。
  6. 「アカウントを削除しています...」の表示後、削除が完了するとログイン画面へ戻ります。

確認ダイアログでは、次の内容が表示されます。

  • アカウント情報は削除されます。
  • 投稿した回答は「削除済み」となり、内容は閲覧できなくなります。
  • ランキングや成績は、サービスの整合性維持のため匿名化して保持される場合があります。
  • この操作は取り消せません。

再認証に失敗した場合は削除処理は実行されません。削除処理中にエラーが発生した場合は、時間をおいて再度お試しください。

削除される情報

アカウント削除時、運営者は次の情報を削除または無効化します。

1. 認証情報

  • Firebase Authentication 上のアカウント（削除）

2. プロフィール情報（匿名化・削除）

  • プレイヤー名（「退会済みプレイヤー」に置き換え）
  • プッシュ通知配信用の FCM トークン（削除）
  • 通知判定用の内部状態（notificationState）（削除）

3. 回答情報（匿名化）

  • 投稿した回答の本文（「この回答は削除されました」に置き換え）
  • 回答に紐づくプレイヤー名（「退会済みプレイヤー」に置き換え）
  • 回答の削除済みフラグ（deleted）の付与

4. 評価キュー

  • ユーザーに紐づく evalQueue データ（削除）

5. アプリ内の端末ローカルデータ

  • 通知履歴
  • お知らせの既読状態

削除後も保持される情報

サービスの整合性維持および運営上の必要性に基づき、次の情報は匿名化したうえで保持される場合があります。

1. ユーザーデータ（Firestore）

users コレクションのドキュメントは削除されず、退会済みフラグ（deleted）および退会日時（deletedAt）が付与された状態で保持されます。プレイヤー名は「退会済みプレイヤー」に置き換えられます。

なお、レーティング・ランク・プレミアム会員フラグ（isPremium）・通算戦績（lifetimeStats）・通知設定（notificationSettings）など、上記の匿名化・削除処理の対象外のフィールドは、そのまま保持されます。確定ランキング（finalTop10）およびシーズン成績（seasonResults）に表示されていたプレミアム表示のみ、false に更新されます。

2. 回答データおよび集計情報

回答ドキュメント自体は削除されません。回答本文およびプレイヤー名は匿名化されますが、評価数・閲覧数・スコア・順位などの集計データ（answerStats を含む）は保持されます。

3. ランキング情報（finalTop10）

お題の確定ランキング（finalTop10）に掲載されている該当回答について、著者名および回答本文は匿名化されます。プレミアム表示（isPremium）は false に更新されます。順位その他の成績情報は保持されます。

4. シーズン成績（seasonResults）

シーズン終了時の確定成績について、プレイヤー名は「退会済みプレイヤー」に置き換えられ、プレミアム表示（isPremium）は false に更新されます。順位・ポイント・ランクその他の成績情報は保持されます。

削除後の注意事項

  • アカウント削除後、削除したデータを復元することはできません。
  • アカウント削除は取り消すことができません。
  • Firebase Authentication 上のアカウントは削除されるため、同じ Google アカウントで再度本サービスを利用する場合は、新たにアカウント登録が必要です。
  • 削除処理は Cloud Functions（deleteAccount）によりサーバー側で実行されます。通信環境等により処理に時間がかかる場合や、処理に失敗する場合があります。
  • 本ページは Web 上の削除申請窓口ではありません。アカウント削除は必ずアプリ内から行ってください。
